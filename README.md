# aroa-ui

A UI Component library made with React

## Installation

- With NPM

```
$ npm install @fsjs/aroa-ui
```

- With Yarn

```
$ yarn add @fsjs/aroa-ui
```

## Usage

```
import { Button } from '@fsjs/aroa-ui'

const MyComponent = () => (
  <div>
    <Button />
  </div>
)
```
